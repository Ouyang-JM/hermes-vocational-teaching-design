# hermes-vocational-teaching-design

> 中职计算机/数媒专业课教学设计撰写技能 —— Hermes Agent Skill

## 适用场景

专为中职计算机应用、数字媒体技术、UI设计、网页设计与制作、Python程序设计、图形图像处理等专业课教师设计。

## 技能亮点

- 🔵🟠 **双模板**：新课（岗位驱动 + AI辅助点） / 技能高考复习课（考纲驱动 + 模拟演练）
- 📐 **四维素养目标**：知识目标 → 能力目标 → 素养目标（对接2025版《职业教育专业教学标准》）
- 🎓 **4+1 课型子模板**：Python编程 / 网页制作 / UI设计 / 图形图像（新课4种 + 复习课1种）
- 🧭 **岗课赛证对接**：技能大赛真题改编 + 考纲分值分布
- 🌱 **课程思政自然融入**：任务情境中内化，不贴标签

## 快速使用

在 Hermes Agent 中说：

> "帮我写一份Python条件语句的教学设计，新课"
> "写一份网页制作——CSS盒子模型的复习课教案"

技能自动按模板输出完整教学设计。

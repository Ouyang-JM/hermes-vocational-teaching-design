# 职业教育教学目标的术语演变

> 2025年研究结论。来源：2025版《职业教育专业教学标准》、2020版《中等职业学校公共基础课程标准》等权威文件。

## 术语演变时间线

```
2001  基础教育"三维目标"：知识与技能 → 过程与方法 → 情感态度与价值观
       └─ 但这是K-12框架，职业教育从未直接搬用
       
2010s 职业教育实践中的主流表述：
      知识目标 → 能力/技能目标 → 素质目标
      └─ 职业教育版"三维"，强调"能力本位"而非"过程体验"

2019  "职教20条"发布 → 进一步强化"能力本位"导向

2020  《中等职业学校公共基础课程标准》发布
      └─ 公共基础课全面转向"学科核心素养"框架
      └─ 如中职语文提出4个核心素养维度

2025  新版《职业教育专业教学标准》发布
      └─ 专业课教学目标的表述："知识+能力+素养"三维仍是主流
      └─ "核心素养"成为关键词 → 第三维从"素质"向"素养"迁移
```

## "素质目标" vs "素养目标"：关键区别

| 维度 | 素质目标（旧） | 素养目标（新·推荐） |
|------|-------------|----------------|
| **语义侧重** | 静态的品质、资质 | 内化后的能力品格 |
| **典型表述** | "培养职业素质" | "形成数字化学习素养" |
| **政策对齐** | 2010s的表述习惯 | 2025版专业教学标准"核心素养"导向 |
| **与思政的关系** | 常常和思政混在一起 | 素养内含思政，不单独加"思政目标"列 |

## 当前主流做法

### 方案A：三维（推荐）
```
知识目标 → 能力目标 → 素养目标
（课程思政内化在素养目标中，各环节自然融入）
```

### 方案B：三维+思政显性化（部分学校使用）
```
知识目标 → 能力目标 → 素养目标 → 思政目标
⚠️ 争议：容易导致"贴标签式"思政融入
```

### 方案C：中职公共基础课的"学科核心素养"框架
```
不再使用三维框架，直接列出4个核心素养维度
⚠️ 仅适用于语文、数学、英语等公共基础课，不适用于专业课
```

## 本文技能的选择

本技能采用 **方案A：知识目标 + 能力目标 + 素养目标**，原因：

1. 对接2025版专业教学标准的核心素养导向
2. 课程思政自然内化在素养目标中，不单独列"思政目标"避免贴标签
3. 专业课领域仍以三维框架为主流，核心素养框架目前仅覆盖公共基础课
4. 原文模板中的"素质目标"已全局更新为"素养目标"

## 注意事项

- 用户的原始 docx 模板中写的是"素质目标"，这是历史习惯
- 生成的教案统一使用"素养目标"
- 如果用户要求严格按原模板输出（含"素质"），则尊重用户选择
- 素养目标的行为动词示例：养成、形成、树立、内化、建立
